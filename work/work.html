I was able to complete this simple Rock, Paper, and Scissors game from planning ahead in pseudocode, divide the problem into subproblems and solve them one by one individually. The application coded using JavaScript. I was able to put my skill to practice on creating conditional functions, how to handle event listeners and when to implement .forEach and .map. .forEach is used when we want to loop over an array and .map is to use when we want to transform a new returned array. I was also able to practice using DOM object to link other files to JavaScript’s file.

This application took a little longer since I tried to add a little more functionalities to it from requirements on The Odin Project. I was able to implement object orientation to store states from event listeners to get the drawing functionality to work. I used a loop to create the grid and set the target value of the slider to get the grid size to change according to the user's selection. The grid is created by using grid-template-rows and grid-template-columns in CSS. After completing this project, I learned the importance of writing reusable code and giving clear variable names. Some functions require the same action like a written function, when they are separated out, it is easy to just plugin and implement the arguments accordingly.
